## النظام الحالي (مكتمل ويعمل)
1. **فرز يومي 10ص السعودية** (cron `0 7 * * 2-6`) → قائمة **A/B**.
2. **14 بوابة** (M1–M14). الأساسية الصارمة (هوية الارتكاز) = **M1-M5** + أرضية RSI.
   - A = صفر نواقص تأكيد · B = ≤3 نواقص · أكثر = رفض.
3. **قائمة ثابتة دائمة**: تُضاف لها الجديد، **لا تُحذف إلا بستوب/هدف** (لا ترفرف).
   التجديد الكامل = الجمعة فقط (`should_renew`). `run_daily_watchlist` = فرز+دمج.
4. **قائمة مراقبة الارتداد** (مستقلة، `wl["pullback"]`): ارتكاز ارتفع فوق دخوله →
   متابعة يومية + تنبيه لحظي عند نزوله للدعم. مرتّبة بالجودة، حد 15.
   - `pullback_live.py` + workflow `pullback_monitor.yml` كل 30د بالسوق.
5. **ترقية B→A** تلقائية + مؤشر صحة بيانات + retry تحميل (لا اختفاء عشوائي).

## قرارات فيصل المحسومة (لا تغيّرها بدون سؤال — موثّقة بالصور)
- الوقف **~7% تحت الدعم**: `STOP_BELOW_LOW_PCT=(5,7)` + `USE_ATR_STOP=False`.
  ⚠️ ATR للوقف كان خطأي (يعمّقه لـ18%) — أُلغي. فيصل 5-7% (ENPH/VMAR).
- MACD **12/26/9** · EMA **30/50** · Fibonacci أهداف (`USE_FIB_TARGETS=True`).
- RSI: قاع ≤27 مثالي (`RSI_OS_HARD=32` أرضية، 27-32 نقص) · الآن ≤40
  (`RSI_MAX_NOW=40`، 40-50 نقص، `RSI_NOW_HARD=50` رفض فوق 50).
- الدخول **دفعات** (فيصل @kisar_): `ENTRY_TRANCHES=3` عند الدعم وصعوداً بخطوة
  `ENTRY_STEP_PCT=3`% (مثال فيصل 1.70/1.75/1.80 وقف 1.50). الوقف 7% تحت الدعم.
- العائد/المخاطرة من **أعلى دفعة** (entry_hi=أسوأ تعبئة) لا السعر الحالي.
- فلوت <50م (`FLOAT_GATE_MAX`) · شورت **≤40ألف** (`SHORT_GATE_MAX`؛ SPPL 35) · ثبات 3-8.
- **فحص أخبار آلي** `scan_news_risk`: طرح/تخفيف/شطب من عناوين ياهو → تحذير بالبطاقة.
- **منظومة 4س** `four_hour_levels` (طبقة مساندة، `fetch_4h` بـprepost): دعوم/أهداف
  من رؤوس/ذيول الشموع الحمرا + «مقاومة↗دعم» + ذيل المسح → سطر «🕓 4س».
- **A→B عند التخفيف**: `news_risk`+كسر الدعم/قاع أدنى → B (في `check_promotions`)،
  يرجع A عند الاستقرار فوق الدعم.
- `SCORE_MIN=45` · `WATCH_MAX_FAILS=3` · `MIN_PRICE=1.5` · `NEAR_PCT=50` · `READY_PCT=75`.
- تحذير الصين/هونغ كونغ (`HIGH_RISK_COUNTRIES`) — تحذير فقط (تظل تظهر).
