# Super_Stocks — فارز أسهم الارتكاز (منهجية فيصل @kisar_)

بوت تيليجرام يفرز **أسهم الارتكاز** في ناسداك آليًا: سهم انفجر ≥100% ثم انهار
≥50% → فلوت صغير + شورت منخفض → يتجمّع الآن عند قاع متشبّع بفجوات فوقه =
مرشّح انفجار. يعمل على GitHub Actions (فرز يومي + تجديد أسبوعي بعد إغلاق
الجمعة + مراقبة لحظية + رادار انطلاق حي بـPolygon).

- **الدليل الكامل للمشروع والقرارات المحسومة:** `CLAUDE.md`
- **ذاكرة الاستمرارية بين الجلسات:** `HANDOFF.md`
- **المنهجية الموثّقة من صور فيصل:** `FAISAL_METHODOLOGY_NOTES.md`
- **الاختبارات:** `python3 test_bot.py` (تعمل بلا إنترنت — البوابة: خروج 0،
  وتعمل آليًا على كل دفع عبر `tests.yml`)

> ⚠️ فارز آلي حسب منهجية موثّقة — ليس توصية استثمارية. القرار النهائي
> وإدارة المخاطر على المستخدم.
